# Biblioteca Salesianos Cartagena

Sistema de gestión de biblioteca con códigos QR para estudiantes.

## Características

- Login seguro para estudiantes
- Generación de códigos QR únicos cada 5 minutos
- Escáner QR para agregar libros
- Gestión de libros en posesión
- Sistema de penalizaciones por retraso
- Responsive para móviles

## Uso

1. Abre `index pagina web.html`
2. Haz clic en "Haz click aqui"
3. Ingresa credenciales
4. Usa el escáner QR para agregar libros

## Tecnologías

- HTML5
- CSS3
- JavaScript
- API de cámara para escáner QR
